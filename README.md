### 📌 Project Overview

A rule-based AI chatbot using deterministic logic and hash map (dictionary) for O(1) response lookup.

### ✅ Features Implemented

- ✅ Handles greetings (hello, hi, hey, good morning, good evening)
- ✅ Handles exit commands (bye, exit, quit, end, terminate, stop, close)
- ✅ Continuous infinite loop (while True)
- ✅ Input sanitization (lowercase, strip, remove punctuation)
- ✅ Hash map dictionary for O(1) lookup
- ✅ Professional fallback responses
- ✅ Conversation logging
- ✅ Colorized terminal output
- ✅ Conversation history viewer
